# 🍓 老司機評測報告生成器 V1.5 (PWA)

這是一個專門為老司機社群設計的「評測文生成工具」。透過視覺化的 UI 介面，讓使用者能快速勾選各項實戰指標，並一鍵產出排版精美、內含 Emoji 的評測報告，方便轉貼至論壇或社群群組。

## ✨ 核心功能
- **視覺化膚色矩陣**：完整 20 格四階膚色選擇，產出精確 HEX 色號。
- **智能店家系統**：預設主流店家名單，自動帶出對應的 JKF 討論區網址。
- **全方位實戰指標**：包含女友感、殘廢澡、按摩技巧、包覆感、私密處特徵等多樣化選項。
- **PWA 行動裝置優化**：支援「加入主畫面」，安裝後可離線使用，具備原生 App 般的流暢體驗。
- **一鍵複製**：產出的報告自動進行空行排版與清單化，適合直接貼文。

## 🚀 快速開始 (安裝教學)

由於本工具支援 PWA 技術，您可以將它安裝在手機桌面上：

### iOS (iPhone/iPad)
1. 使用 **Safari** 瀏覽器開啟本工具網址。
2. 點擊瀏覽器底部的 **「分享」** 按鈕 (向上箭頭圖示)。
3. 向下捲動選單，點擊 **「加入主畫面」**。

### Android
1. 使用 **Chrome** 瀏覽器開啟網址。
2. 點擊網址列右側的 **「三個點」** 選單。
3. 點擊 **「安裝應用程式」** 或 **「加入主畫面」**。

## 🛠️ 技術架構
- **HTML5 / CSS3**：採用響應式卡片設計，適配各型號手機。
- **JavaScript (Vanilla)**：無須外部套件，輕量且快速。
- **Service Worker**：提供資源快取，支援離線操作。
- **Web App Manifest**：定義 App 外觀、圖示與啟動模式。

## 📂 檔案清單
- `index.html`: 主程式與使用者介面。
- `manifest.json`: PWA 設定檔。
- `sw.js`: Service Worker 離線邏輯。
