# 🍓 老司機評測報告生成器 V1.55 (PWA)

這是一個專門為老司機社群設計的「評測文生成工具」。透過高度直覺的 UI 介面，讓使用者能快速勾選實戰指標、自定義心得樣式，並一鍵產出具備專業量化數據（Metrics）的評測報告，方便轉貼至論壇或社群群組。

## ✨ V1.56 更新
- 🎨 整合式心得面板：將 K/H/S 諮詢、心得內容、字型樣式控制與 Emoji 工具列整合至單一面板，優化操作動線。
- 😃 圖像化 Emoji 列：
- 新增表情類圖示：😭, 😂, 😡, 😢, 🤢。
- 新增性愛與氣氛類圖示：🍆, 🍑, 🛌, 🏩, 🔥, 🧨 等。
- 📌 K/H/S 欄位優化：提示文字更新為「例如：請跟店家洽談」，並支援與心得同步套用視覺樣式。

🥛 細節選項補強：奶軟度選項包含「前軟後扎實」，更貼近真實體驗描述。

## ✨ V1.55 重大更新
- 🎨 心得樣式自定義：支援針對「服務互動心得」進行字型粗細、大小（14px - 32px）與顏色的調整。
- 😃 Emoji 快捷列：內建 30 個精選常用 Emoji，支援在游標處精確插入，豐富心得內容。
- 🏆 專業量化指標 (Metrics)：引進綜合評級系統（SSS 至 C 級）與 CP 值（1.0 - 10.0）評分，取代模糊的文字評價。
- 📌 K/H/S 專屬欄位：新增諮詢項目欄位，預設為「請跟店家接洽」。
- 🥛 細節優化：奶軟度選項新增「前軟後扎實」，更貼近真實體驗描述。

## ✨ 核心功能
- **視覺化佈局**：
    - 電腦版：採用雙欄並排設計，左側輸入資料，右側即時預覽與操作。
    - 手機版：自動切換為 RWD 單欄卡片佈局，適合單手操作。
- **視覺化膚色矩陣**：完整 20 格四階膚色選擇，產出精確 HEX 色號。
- **智能店家系統**：預設主流店家名單，自動帶出對應的 JKF 討論區網址。
- **全方位實戰指標**：包含女友感、殘廢澡、按摩技巧、包覆感、私密處特徵等多樣化選項。
- **PWA 行動裝置優化**：支援「加入主畫面」，安裝後可離線使用，具備原生 App 般的流暢體驗。
- **高品質匯出**：
    - HTML 下載：以「妹名」自動命名檔案，並產出大字體、保留樣式與 Icon 的 HTML 報告檔。
    - 一鍵複製：產出的報告自動進行空行排版與清單化，適合直接貼文。

## 🚀 快速開始 (安裝教學)
由於本工具支援 PWA 技術，您可以將它安裝在手機桌面上：

### iOS (iPhone/iPad)
1. 使用 **Safari** 瀏覽器開啟本工具網址。
2. 點擊瀏覽器底部的 **「分享」** 按鈕 (向上箭頭圖示)。
3. 向下捲動選單，點擊 **「加入主畫面」**。

### Android
1. 使用 **Chrome** 瀏覽器開啟網址。
2. 點擊網址列右側的 **「三個點」** 選單。
3. 點擊 **「安裝應用程式」** 或 **「加入主畫面」**。

## 🛠️ 技術架構
- **HTML5 / CSS3**：採用響應式卡片設計，適配各型號手機。
- **JavaScript (Vanilla)**：無須外部套件，輕量且快速。
- **Service Worker**：提供資源快取，支援離線操作。
- **Web App Manifest**：定義 App 外觀、圖示與啟動模式。

## 📂 檔案清單
- `index.html`: 主程式與使用者介面。
- `manifest.json`: PWA 設定檔。
- `sw.js`: Service Worker 離線邏輯。
- `icon.png`: 應用程式圖示 (建議為 512x512)。
